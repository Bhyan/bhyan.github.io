Treinamento automação de teste com HTTParty

2024-04-16 Um minuto de leitura
Teste • Api • Httparty • Ruby

Recentemente precisei automatizar os testes de uma API em um cliente que estava alocado, a stack dele era toda baseada em Ruby, então a equipe optou por utilizar as bibliotecas HTTParty e RSpec.

Ao finalizar o projeto ficou o conhecimento, então tive a ideia de socializa-lo, assim criei um treinamento que foi ministrado a aquipe interna da empresa onde atuo utilizando a mesma tecnologia utilizada no cliente, para o treinamento usamos a API testada foi a ServeRest.

Aqui estou centralizando os slides e o código do treinamento. Se conseguir autorização para colocar no YouTube o vídeo do treinamento eu adiciono aqui posteriormente.
